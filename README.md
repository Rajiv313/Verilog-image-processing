FPGA-Based Image Processing Using Verilog HDL

This project implements a basic image processing system on FPGA using Verilog HDL. The design reads a bitmap image, performs operations such as brightness adjustment (positive and negative values) and image inversion, and writes the processed image back to a BMP file format. The system was verified through simulation using a dedicated testbench, demonstrating hardware-oriented image processing suitable for FPGA and digital design applications.

Key Features

Image read/write modules for BMP file handling

Brightness enhancement and reduction operations

Image inversion (negative image generation)

Parameterized Verilog design for easy modification

Simulation and verification using waveform analysis

Tools & Technologies

Verilog HDL • FPGA Design • Xilinx Vivado/ModelSim • Digital Image Processing

This project demonstrates the implementation of hardware-based image processing algorithms, highlighting skills in Verilog coding, FPGA simulation, memory handling, and digital system design.
